What is a KWIC?

KWIC (= Keywords in Context) is the most common format for concordance lines and shows the context of a word within a text. It is often displayed as a table like the one on the left. This table shows the keywords (keywords in table), the text immediately before the keyword (pre in table) and immediately after the keyword (post in table), and often also the places/indices where the keyword occurs in the text (from and to in table). The latter is especially interesting for phrases or sentences.

How to use

The KWIC table of a play or selected plays of an author are calculated here. For the latter, the checkbox Select multiple plays must be selected in the panel on the left. General options such as stop word consideration, punctuation removal and lowercase conversion of words can be selected in the left panel. Note: Here, the checkbox Remove stopwords only refers to the listing of stopwords in the keyword list. No stopwords are removed from the pre and post columns. Special options like the window size (number of words before/after the keyword), the sorting method (sorting of the available keywords by an alphabetical order or by the most frequent words in descending order according to the largest frequency, shown in parantheses behind the word) or the keyword selection can be found in the lower left panel.

The KWIC table on the left consists of five columns:

  • docname: Name of the play. Especially important to distinguish the plays when multiple plays with the option Select multiple plays are selected. The option is greyed out if only one play is selected. It may happen that this option is greyed out, even if several plays have been selected. This means that the selected keyword only appears in one play. Plays that do not contain the keyword are not shown here.
  • from/to: Place in the text where the keyword begins and ends. Since no phrases are supported in this tool, from and to are identical.
  • pre: The phrase/sentence immediately before the keyword.
  • keyword: The selected keyword.
  • post: The phrase/sentence immediately after the keyword.

It is possible to search for specific column values in the table using the search masks under the respective column names or for values in the whole table using the search bar at the top. Show n entries can be used to increase the number of table entries per page. Below the table, information on all entries is given and a page turning function is provided.

Note: For authors with multiple plays, it may take a short time to calculate the KWIC table.
